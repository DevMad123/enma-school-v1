# Services

Ce dossier est destiné aux classes de services de l'application Enma School.

## À propos
Les services contiennent la logique métier réutilisable et les opérations complexes 
qui ne relèvent ni des modèles ni des contrôleurs.

## Règles d'utilisation
- Un service par fonctionnalité métier principale
- Injection de dépendances via le constructeur
- Méthodes avec types de retour explicites
- Documentation complète des méthodes publiques

## Exemples futurs
- `StudentService` : Gestion des étudiants
- `ClassroomService` : Gestion des salles de classe
- `GradeService` : Gestion des notes
- `AttendanceService` : Gestion des présences

**Note :** Ce dossier est actuellement vide et sera rempli lors du développement des fonctionnalités métier d'Enma School.
